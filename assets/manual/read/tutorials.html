Video Tutorials

You can find the latest video tutorials on our youtube channel.

Part 1: Content and Components Review

This video shows you how to get started and takes you through the process of using the framework. This video will take you through everything from downloading the tools to using the styles and pieces in the block design format.

Part 2: How to Start and Create Website with Startup Design Framework

How to stack several blocks together in Startup Framework to make a website.

Part 3: How to work with Less in Startup Framework

In this video tutorial, we'll show you how to use the LESS to compile code for a website. We explain how to find the appropriate .less files to use and how to get your code set up properly.

How to Customize a Website with Startup Framework (Part 1 - Headers)

This video takes you through customizing the header and the basic website framework. With just a few simple tweaks in the HTML, you can convert the framework template to a custom site. The 6-minute video walks you through changing the header, text elements and background image.

How to Customize a Website with Startup Framework (Part 2 - Images)

This video takes you in depth for customization of images. In addition to setting a new image, you can also change the opacity and other settings including the size and scale of the image. You can also customize the way an image looks on the site, as either a smaller image that works with a background or a full-size image that fills the screen. The 6-minute video walks you through setting a new background or main image, image settings and setting styles for hover and navigation links.

How to Customize a Website with Startup Framework (Part 3 - Content)

This video teaches you how to create custom styles for the content section of the website using Startup Framework. Customization options include changes to the color palette, padding styles or icon styles. The 6-minute video walks you through changes to content blocks within the framework. Using CSS styling, background, fonts, colors and location of elements can be moved on the site.

How to Customize a Website with Startup Framework (Part 4 - Footer, Parallax, Typekit)

This video explains how to customize the website footer and even add parallax effects to your custom-built site using Startup Framework. (It's easier than you might think!) The tutorial also helps you understand and troubleshoot some common issues. Further, you will learn how to customize typography options using Adobe Typekit for an even more custom website design. The 8-minute video walks you through adding some fancier "tricks" to your custom website, including trendy parallax scrolling and custom fonts.

How to Work with Generator

How to use the Generator for making the process of creating website templates easier.

How to Add, Move, Remove the Blocks Manually in the Generator

Put a few blocks from Startup Framework in the design created by the Generator (if you would want to do that later manually), or how to get rid of any of the blocks locally, if you don't need it any more.
